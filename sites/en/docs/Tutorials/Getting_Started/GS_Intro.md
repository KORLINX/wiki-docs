# Getting Started

New to KORLINX products? These guides walk you through everything from unboxing to your first data transmission.

## What You'll Need

- A KORLINX module or evaluation kit
- USB-A to USB-C cable (or UART-to-USB adapter)
- Terminal emulator: [Tera Term](https://ttssh2.osdn.jp/), [PuTTY](https://putty.org/), or the VS Code Serial Monitor extension
- Active SIM card (for cellular products)

## First Steps

1. **Connect** the module to your computer via USB or UART
2. **Open** a serial terminal at the module's default baud rate (typically 115200)
3. **Send** `AT` — the module should respond `OK`
4. **Configure** your APN and network settings
5. **Test** connectivity with `AT+PING`

:::note
Network attach steps vary by product. Refer to your product's Network Attach document for the basic connectivity guide.
:::
