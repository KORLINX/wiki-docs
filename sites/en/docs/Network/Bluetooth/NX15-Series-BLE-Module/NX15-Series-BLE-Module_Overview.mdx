# NX15 Series BLE Module Overview

{/* Image suggestion: NX15 module product image showing both chip antenna and u.FL antenna variants. Suggested path: /img/bluetooth/nx15/nx15-series-overview.png */}

The **NX15 Series** is a compact Bluetooth Low Energy module designed for low-power IoT products, sensors, beacons, wearables, smart home devices, and embedded wireless applications.

It is based on the Nordic **nRF54L15 SoC** and supports **Bluetooth LE 5.4**. Depending on the software stack, it can also be used for 802.15.4-based applications such as **Thread**, **Zigbee**, and **Matter over Thread**.

This overview provides a short introduction only. For hardware design, antenna layout, low-power optimization, and application development, continue to the dedicated pages listed below.

---

## When to Use NX15

NX15 is suitable when your product needs:

| Requirement | Why NX15 Fits |
| --- | --- |
| Compact wireless module | Small LGA module for space-constrained designs |
| Low-power BLE communication | Suitable for battery-powered IoT devices |
| Sensor or beacon behavior | Supports periodic data transmission and BLE advertising |
| Embedded application development | Works with nRF Connect SDK / Zephyr RTOS |
| Flexible antenna option | Available with chip antenna or u.FL external antenna variants |
| Future multiprotocol path | Can support Thread / Zigbee / Matter depending on software stack |

## Series Positioning

The NX15 Series includes variants for different hardware integration needs.

{/* Image suggestion: Simple model lineup diagram: NX15i = chip antenna, NX15E = u.FL antenna, 32 = external Flash, 00 = no external Flash. Suggested path: /img/bluetooth/nx15/nx15-model-lineup.png */}

| Variant Type | Description |
| --- | --- |
| NX15i Series | Integrated chip antenna variants |
| NX15E Series | u.FL external antenna variants |
| 32 models | Include external SPI Flash |
| 00 models | No external Flash, more focused on compact/full GPIO use cases |

For detailed model selection, use the **NX15 Module Integration / Model Selection** page.
