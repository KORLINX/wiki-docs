# NX40 Module Integration Overview

{/* Image suggestion: High-level diagram showing NX40 mounted on a custom host PCB with power, SWD, reset, GPIO, Flash option, and antenna path. Suggested path: /img/bluetooth/nx40/module-integration/nx40-module-integration-overview.png */}

This section is for developers and hardware engineers who integrate the **NX40 module** into a custom PCB or host system.

It focuses on practical module-level integration topics such as model selection, PCB integration, pad/pin planning, antenna-related design checks, handling, soldering, and first prototype troubleshooting.

:::note
This section is not a general Bluetooth introduction. For BLE concepts, see the Bluetooth Intro page. For product-level usage, see the NX40 Series BLE Module pages.
:::

| User Type | Goal |
