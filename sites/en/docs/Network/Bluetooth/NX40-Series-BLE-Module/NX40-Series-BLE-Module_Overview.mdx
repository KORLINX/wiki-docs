# NX40 Series BLE Module Overview

{/* Image suggestion: NX40 module product image showing chip antenna and external antenna variants. Suggested path: /img/bluetooth/nx40/nx40-series-overview.png */}

The **NX40 Series** is a compact wireless module series for Bluetooth Low Energy and 2.4 GHz embedded applications.

The NX40 Series is based on the **nRF52840** platform and is intended for applications such as BLE sensors, beacons, wearable devices, smart home devices, industrial monitoring nodes, and short-range wireless control systems.

This page provides only a short overview. For first bring-up, hardware design, antenna design, low-power behavior, and BLE application development, continue to the dedicated pages listed below.

---

## When to Use NX40

NX40 is suitable when your product needs:

| Requirement | Why NX40 Fits |
| --- | --- |
| Bluetooth Low Energy communication | Suitable for short-range wireless devices and sensors |
| 2.4 GHz multiprotocol capability | Can support BLE and other 2.4 GHz protocol use cases depending on firmware |
| Compact embedded design | Suitable for integration into custom PCB products |
| Sensor or beacon behavior | Supports periodic data transmission and advertising-based workflows |
| Host board integration | Can be used as a module inside a larger embedded system |
| Low-power IoT direction | Can be used in battery-powered products when firmware and hardware are optimized |

:::note
This page does not provide a full specification table. Detailed power, pin, RF, and production data should be placed in the dedicated hardware and integration pages after the NX40 datasheet is finalized.
:::

## NX40 Series Positioning

The NX40 Series follows the same model structure as the NX15 family: antenna option and external Flash option.

{/* Image suggestion: Simple model lineup diagram: NX40i = chip antenna, NX40E = u.FL external antenna, 32 = external Flash, 00 = no external Flash. Suggested path: /img/bluetooth/nx40/nx40-model-lineup.png */}

| Model | Antenna | Flash | Description |
| --- | --- | --- | --- |
| NX40i32 | Chip antenna | 32 Mbit SPI Flash | Chip antenna variant with external Flash |
| NX40E32 | u.FL external antenna | 32 Mbit SPI Flash | External antenna variant with external Flash |
| NX40i00 | Chip antenna | None | Chip antenna variant without external Flash |
| NX40E00 | u.FL external antenna | None | External antenna variant without external Flash |

:::caution
Confirm the final model list, mechanical data, pin map, and RF layout details against the official NX40 datasheet before production release.
:::
