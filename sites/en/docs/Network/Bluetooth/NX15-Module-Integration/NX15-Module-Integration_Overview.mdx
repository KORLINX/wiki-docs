# NX15 Module Integration Overview

{/* Image suggestion: High-level integration diagram showing NX15 module mounted on a host PCB with power, SWD, GPIO, antenna, and host MCU blocks. Suggested path: /img/bluetooth/nx15/module-integration/nx15-module-integration-overview.png */}

This section is for developers and hardware engineers who integrate the **NX15 module** into a custom PCB or host system.

It focuses on practical module-level integration topics such as model selection, PCB integration, LGA pad layout, handling, soldering, and bring-up troubleshooting.

:::note
This section is not a product introduction page. For general product information, see the NX15 Series BLE Module documentation.
:::

---

## Who Should Read This Section?

Read this section if you are:

| User Type | Goal |
| --- | --- |
| Hardware engineer | Integrate NX15 into a custom PCB |
| Firmware engineer | Understand available pins, debug access, and board constraints |
| Production engineer | Prepare SMT assembly, handling, and reflow process |
| System integrator | Validate power, antenna, programming, and bring-up path |
