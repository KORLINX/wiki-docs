# KSE-91A Quick Start

The KSE-91A is a cellular communication module based on the **nRF9151** platform. This guide helps you quickly understand the LCD display, one-button operation, connection states, Location screen, Diagnostics pages, and host interface configuration.


## Hardware Overview

The KSE-91A hardware provides an external LTE/GNSS antenna connection, a host equipment connector, a front LCD display, and a single button for screen navigation and configuration.

## Startup and Connection Screens

<img
  src="/img/KSE-91A/screen.png"
  alt="KSE-91A screens"
  className="blog-first-image"
/>

The module displays several screens during power-on and cellular network attachment.

| Screen | Meaning | User Action |
| --- | --- | --- |
| Boot | Module is powering up and shows product branding, platform, and firmware version | No action needed |
| Starting | Modem is being initialized | Wait |
| Connecting | Module is searching for and attaching to the mobile network | Wait; this may take up to about one minute |
| Connected | Module is online and attached to the network | Normal operating state |
| No Signal | No mobile coverage is available | Move the module or antenna to a better reception area |
| Check SIM | SIM card is not detected | Power off, reseat SIM, then power on again |
| Service Error | Network reached, but service was refused or failed | Check SIM plan/service and collect Diagnostics information |

## Button Operation and Screen Actions

KSE-91A uses a single button. The button behavior depends on the screen currently shown on the LCD.

| Action | How to Perform | General Meaning |
| --- | --- | --- |
| Short press | Press and release quickly | Move to the next screen or next selectable item |
| Long press | Press and hold for about 2 seconds | Enter, select, or exit depending on the current screen |
| Hold at power-on | Hold the button while powering on the module | Enter Interface Configuration |

:::note
The button action is **context-based**. Always check the current screen before pressing the button.
:::

### Action on Current Screen

| Current Screen / Mode | Short Press | Long Press | Note |
| --- | --- | --- | --- |
| Home / Connected | Switch to Location | Open Diagnostics | Main operating screen |
| Location | Switch back to Home / Connected | Open Diagnostics | Shows GPS information |
| Diagnostics | Move to the next Diagnostics page | Exit Diagnostics | Last page loops back to the first page |
| Interface Configuration | Move to the highlight to the next option | Select the highlighted option | Show press never commits a change |
| Fault screen, such as Check SIM or Service Error | No required | Open Diagnostics if available | Fault screens appear automatically |
| No input in Diagnostics | - | - | Automatically exits after about 30 seconds |


:::note
Fault screens interrupt automatically. If a problem occurs, such as **Check SIM** or **Service Error**, the module shows the fault screen regardless of which screen you were viewing.
:::

### Diagnostics

<img
  src="/img/KSE-91A/diagnostics.png"

:::tip
Action on this screen
- To open **Diagnostics**, **long press** from either **Home / Connected** or **Location**.

Diagnostics provide **read-only** technical values for installers and support.

#### Entering Diagnostics

| To reach | Action |
| --- | --- |
| Diagnostics | Long press from Home / Connected or Location |
| Next Diagnotics page | Short press while in Diagnostics |
| Exit Diagnostics | Long press while in Diagnostics |
| Automatic exit | No input for about 30 seconds |

:::note
Diagnostics are read-only. Opening Diagnostics does not change any module settings.
:::

#### Action in Diagnostics

| Current Diagnostics Page | Short Press | Long Press |
| --- | --- | --- |
| Network | Move to Signal page | Exit Diagnostics |
| Signal | Move to Link page | Exit Diagnostics |
| Link | Move to Device page | Exit Diagnostics |
| Device | Return to Nextwork page | Exit Diagnostics |


#### Diagnostics Pages

| Diagnostics Page | Information Shown |
| --- | --- |
| Network | Registration state, operator, cellular type, band |
| Signal | RSRP, RSRQ, SNR, Cell ID |
| Link | IP, protocol, data state, sent/received data |
| Device | IMEI, ICCID, host UART, firmware version |

:::info
Use Diagnostics when you need to collect technical information for installation, field support, or troubleshooting.
:::

#### Interface Configuration
Interface Configuration is used to select the host communication interface.
To enter this mode, **hold the button while powering on the module.**

<img
  src="/img/KSE-91A/KSE-91A_interface.png"
  alt="KSE-91A-interface"
  className="blog-first-image"
/>

#### Action in Interface Configuration

| Current Screen | Short Press | Long Press |
| --- | --- | --- |
| Interface option list | To select the next option | To confirm selection |

A **short press never commits an interface change**. In Interface Configuration, only a deliberate **long press** selects the highlighted option.

#### Available options

| Option | Meaning |
| --- | --- |
| RS485 | Use RS485 host interface |
| RS232 | Use RS232 host interface |
| UART | Use UART host interface |
| Cancel | Exit without changing anything |


After selecting a new interface, the module displays a confirmation screen and restarts.

:::caution
After changing the module interface, configure the host equipment to use the same interface. If the host and module interfaces do not match, communication will not work.
:::

## Troubleshooting

| Screen / Symptom | Meaning | Recommended Action |
| --- | --- | --- |
| No Signal | No mobile coverage | Move to a better reception area and check the antenna |
| Check SIM | SIM card is not detected | Power off, reseat SIM, then power on again |
| Service Error | Service refused or unrecoverable network issue | Check SIM service plan and collect Diagnostics information |
| GPS stays searching | GPS has not acquired a fix | Wait longer and improve antenna/sky visibility |
| Cannot communicate with host | Interface mismatch | Re-enter Interface Configuration and select the correct RS485/RS232/UART setting for the host |
| Accidentally entered Diagnostics | Long press was triggered | Long press again to exit, or wait for automatic exit |
