## Introduction

KSE-91A is an industrial LTE-M and NB-IoT modem designed for reliable, low-power IoT connectivity in demanding environments. As part of the KORLINX industrial LTE-M product family, the device provides a ready-to-use external modem solution that enables rapid deployment of IoT applications without requiring complex hardware design or RF expertise.

With support for LTE Cat-M1 and NB-IoT networks, integrated GNSS positioning, and multiple industrial communication interfaces, KSE-91A simplifies the connection of sensors, controllers, and embedded systems to cloud services and enterprise platforms.

The compact enclosure, external antenna design, and AT command-based control interface make KSE-91A suitable for industrial automation, remote monitoring, smart logistics, asset tracking, and smart city deployments.

## Why Choose KSE-91A?

KSE-91A is designed for organizations that need fast and reliable cellular connectivity without developing a custom LTE hardware platform.

Compared with embedded modem modules, KSE-91A provides:

- Faster deployment
- Easier installation
- Better compatibility with existing equipment
- Simplified maintenance and replacement
- Reduced development effort

The device can be integrated into both new and existing systems through standard serial interfaces, making it ideal for rapid proof-of-concept projects as well as large-scale industrial deployments.


## Typical Applications

KSE-91A can be applied across smart industry environments where reliable cellular connectivity and real-time data communication are required.

Typical use cases include:

- **Smart agriculture** — remote monitoring and field data collection
- **Smart logistics** — vehicle, cargo, and asset tracking
- **Smart industry** — connect industrial equipment connectivity and predictive maintenance
- **Construction and infrastructure** — field equipment monitoring and real-time site visibility
- **Remote Monitoring** — enable secure remote access to utility meters, kiosks, and infrastructure equipment

These applications help organizations collect data, monitor operational status, and connect distributed assets to cloud-based management platforms.


## Technical Highlights

| Item | Specification |
|--------|--------|
| Cellular Network | LTE Cat-M1 / NB-IoT |
| MCU | 64 MHz Arm Cortex-M33 |
| Memory | 1 MB Flash + 256 KB RAM |
| External Flash | 4 MB |
| GNSS | Supported |
| Input Voltage | 5–28 VDC |
| SIM | Nano SIM |
| LTE Antenna | External SMA |
| Protocols | MQTT, CoAP, HTTP, LwM2M |
| Firmware Update | FOTA via LTE |
| Security | Arm TrustZone + CryptoCell 310 |
