The following command groups will be documented here:

| Category | Commands |
|----------|---------|
| General | `AT`, `ATI`, `AT+CFUN`, `AT+CPIN`, `AT+CMEE` |
| Network | `AT+CEREG`, `AT+COPS`, `AT+CSQ`, `AT%CESQ`, `AT%XSYSTEMMODE` |

| TCP/UDP sockets | `AT#XSOCKET`, `AT#XCONNECT`, `AT#XSEND`, `AT#XRECV`, `AT#XCLOSE` |
