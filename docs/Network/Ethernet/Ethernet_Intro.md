# Ethernet Overview

KORLINX Ethernet solutions provide deterministic, high-reliability wired connectivity for industrial IoT — from standard Fast/Gigabit Ethernet to industrial protocols with Power over Ethernet (PoE).

## Ethernet for Industrial IoT

| Standard | Speed | Key Feature | Use Case |
|----------|-------|-------------|----------|
| **100BASE-TX** | 100 Mbps | Universal compatibility | Factory sensors, PLCs |
| **1000BASE-T** | 1 Gbps | High throughput | Gateways, HMIs |
| **IEEE 802.3af** (PoE) | 15.4 W delivery | Power + data over one cable | IP cameras, access points |
| **IEEE 802.3at** (PoE+) | 30 W delivery | Higher power budget | Industrial actuators |
| **Time-Sensitive Networking** | 100M / 1G | Deterministic timing | Motion control, robotics |

---

## Industrial Protocols (Layer 7)

KORLINX Ethernet modules support bridging to common industrial protocols:

- **Modbus TCP** — widely deployed in factory and building automation
- **EtherNet/IP** — CIP-based protocol, common in Allen-Bradley PLCs
- **PROFINET** — Siemens-ecosystem industrial Ethernet
- **OPC UA** — unified architecture for secure, platform-neutral data exchange

---

## Advantages of Wired Ethernet

- **Deterministic latency** — no wireless interference or retransmissions
- **High throughput** — suitable for video, large configuration files, firmware OTA
- **Security** — physical layer security, no wireless eavesdropping
- **PoE** — simplify installations by eliminating separate power runs
